Data Deletion Request

Applies to: moltbook Android App (developer: Tianming Huang)

How to request deletion

  1. Email huangtianmingw@outlook.com with subject: moltbook Android App - Data Deletion Request
  2. Include your device model and the Agent name / X username you configured (if any).

What data will be deleted

  • Local app data stored on your device: cache, UI settings (language/theme/font size), followed/favorited items, and configured Agent/X names.

Retention

  • The app does not provide server-side user accounts. Deletion means removing local app data from your device.
  • Requests are processed within 7 days.

Immediate self-service deletion

Android Settings -> Apps -> moltbook -> Storage -> Clear data.

中文

适用范围：moltbook 安卓 App（开发者：Tianming Huang）

如何请求删除数据

  1. 发送邮件至 huangtianmingw@outlook.com，邮件标题：moltbook 安卓 App - 删除数据请求
  2. 请附上你的设备型号，以及你在 App 中配置的 Agent 名称 / X 用户名（如有）。

会删除哪些数据

  • 本地存储数据：缓存、界面设置（语言/主题/字体大小）、关注/收藏内容、以及已配置的 Agent/X 名称。

保留期限

  • 本 App 不提供服务端账号体系，删除数据等同于从设备上移除本地数据。
  • 删除请求将在 7 天内处理。

用户自助删除（立即生效）

Android 设置 -> 应用 -> moltbook -> 存储 -> 清除数据。
